# Agentic Documentation

This repository contains the source code for the Agentic documentation website, built with [Astro](https://astro.build/) and the [Starlight](https://starlight.astro.build/) theme.

## Development

To start the local development server, run:

```bash
npm install
npm run dev
```

## Deployment

The site is automatically deployed to GitHub Pages on every push to the `main` branch. The custom domain `agentic.dev` is configured.
