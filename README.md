# Agentic Documentation

This repository contains the source code for the Agentic documentation website, built with [Astro](https://astro.build/) and the [Starlight](https://starlight.astro.build/) theme.

## Enterprise Context

Agentic is the AI & Agents division of the [jbcom enterprise](https://jbcom.github.io). This documentation site is part of a coherent package of specialized tools, sharing a unified design system and interconnected with sibling organizations like [Strata](https://strata.game) and [Extended Data](https://extendeddata.dev).

## Development

To start the local development server, run:

```bash
npm install
npm run dev
```

## Deployment

The site is automatically deployed to GitHub Pages on every push to the `main` branch. The custom domain `agentic.dev` is configured.
